# Job Search Criteria

- Product-based companies
- Startups
- SaaS companies
- AI startups
- FinTech companies
- Remote / Hybrid / Delhi NCR / Noida / Gurugram / Bangalore / Hyderabad / Pune
- Preferred stipend: ₹15,000/month or above
- Internship duration: 2–6 months
- PPO preferred
- ATS-friendly hiring process
- Posted within the last 30 days

# Discovered Opportunities

Claude searched internship opportunities using the Indeed Connector and prioritized roles matching my profile based on:

- Technical skills
- Academic background
- Preferred locations
- Internship preferences
- Learning opportunities
- Match score

The analysis included:

- Company Name
- Role
- Location
- Posted Date
- Application Link
- Match Score
- Why the role matches my profile
- Expected Stipend / CTC

# Key Learnings

- ATS-friendly resumes significantly improve visibility.
- Strong projects matter more than simply listing technologies.
- Modern internships heavily prioritize React and Full Stack skills.
