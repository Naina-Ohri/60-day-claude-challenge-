# Day 17 – AI Vehicle Cost & Fuel Analysis Dashboard

## Objective
Build an AI-powered Vehicle Cost & Fuel Analysis Dashboard using Claude by analyzing a CSV dataset and generating an interactive HTML dashboard with business metrics and visualizations.

# Vehicle Details

- **Vehicle:** Tata Nexon
- **Fuel Type:** Petrol
- **Usage:** Mixed
- **Distance:** 1000 km/month
- **Vehicle Age:** 3 Years

---

# Dashboard Highlights

The generated dashboard includes:

- Fuel Cost per Kilometer analysis
- Monthly Fuel Cost estimation
- E85 cost comparison and break-even analysis
- CO₂ emissions comparison
- Vehicle age-wise maintenance analysis
- E85 suitability score
- Fuel comparison cards with advantages and disadvantages
- Interactive SVG charts and responsive dashboard layout

---

# Key Insights

- Petrol provided a balanced combination of running cost and convenience for the selected vehicle profile.
- E85 showed lower fuel cost potential but introduced trade-offs in efficiency and overall running economics.
- Visual dashboards made it much easier to compare different fuel options and understand maintenance trends at a glance.
- KPI cards and charts helped summarize complex CSV data into meaningful business insights.

---

# Observations

- Claude successfully transformed raw CSV data into a professional HTML dashboard.
- The generated dashboard included responsive design, SVG charts, KPI cards, and comparative analysis without requiring manual coding.
- Structured prompts significantly improved the quality and organization of the generated output.

---

# Key Learnings

- AI can analyze structured CSV datasets and automatically generate business dashboards.
- Well-defined prompts produce more accurate analytics and visualizations.
- Interactive dashboards make data interpretation faster and easier than reading raw tables.
- Claude can automate many real-world data analysis and reporting tasks using natural language prompts.
