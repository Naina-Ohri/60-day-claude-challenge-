# Custom Skill

**Skill Name:** brain-dump-action-planner

**Purpose:**
A reusable Claude Custom Skill that converts unstructured notes into organized dashboards containing summaries, action items, risks, blockers, open questions, conflicts, and project insights without inventing or assuming missing information.

---

# Test Scenario

The skill was tested using a sample **project meeting discussion** containing:

- Multiple team members
- Project decisions
- Action items
- Risks
- Blockers
- Open questions
- Pending deadlines

The skill generated an interactive HTML dashboard that organized all information into a clean project management interface.

---

# Dashboard Highlights

The generated dashboard included:

- Executive Summary
- Key Takeaways
- Action Items Table
- Open Questions
- Risks & Blockers
- Conflict Detection
- Additional Notes
- Status Badges
- Responsive Dashboard Layout

---

# Observations

- Claude successfully transformed lengthy meeting notes into a structured dashboard.
- Action items were automatically organized with owners, deadlines, and statuses.
- Risks, blockers, and unresolved questions became much easier to identify.
- The HTML dashboard followed a modern project management style similar to Notion and ClickUp.

---

# Key Learnings

- Custom Skills eliminate repetitive prompting for recurring workflows.
- AI can organize large amounts of unstructured information within seconds.
- Interactive dashboards improve readability compared to raw meeting notes.
- Structured prompts produce consistent and reusable outputs for project planning and collaboration.

---

# Conclusion

Day 18 demonstrated how Claude Custom Skills can automate information organization by converting messy notes into professional dashboards. This workflow can significantly improve productivity for meetings, brainstorming sessions, project management, and personal planning.
