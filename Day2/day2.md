# Day 2 - Prompt Engineering

## Objective

Learn the difference between a simple prompt and an engineered prompt.

## Lazy Prompt

Create an image explaining Prompt Engineering.

## Engineered Prompt

Used a detailed prompt with context, requirements, examples, formatting instructions, and image specifications.

## Key Learnings

* Better prompts produce better AI outputs.
* Context and clear instructions improve response quality.
* Structured prompts help AI generate more relevant and organized content.
* Prompt engineering is an important skill when working with AI tools.
