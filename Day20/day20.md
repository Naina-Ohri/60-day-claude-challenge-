# Day 20 – AI Face Puzzle Game

## Overview
Today I built an interactive **AI Face Puzzle Game** using Claude. The application captures an image using the webcam, converts it into puzzle pieces, and creates a playable drag-and-drop puzzle with different difficulty levels.

## Features Implemented

- Webcam access using `getUserMedia()`
- Live camera preview
- Photo capture
- Puzzle generation from captured image
- Multiple difficulty levels (3×3, 4×4, 5×5)
- Drag-and-drop puzzle interaction
- Touch support for mobile devices
- Timer
- Move counter
- Correct pieces tracker
- Puzzle completion detection
- Leaderboard using Local Storage
- Responsive UI
- Modern glassmorphism design
