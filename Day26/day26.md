# Day 26 – Prior Authorization Workflow Simulator

## Overview
Built an interactive Prior Authorization Workflow Simulator using HTML, CSS, and Vanilla JavaScript to visualize the US healthcare prior authorization process through a gamified drag-and-drop workflow.

---

## Workflow Summary

**Scenario:** Lumbar Spine MRI (Imaging)

**Workflow Stages Completed:**
1. Patient Intake
2. Medical Necessity Review
3. PA Document Collection
4. Submit to Payer
5. Payer Utilization Review
6. Determination

The simulator demonstrates how a healthcare prior authorization request progresses from the patient through the provider to the insurance payer, highlighting each decision point and educational explanation.

## Key Learnings

- Learned the complete Prior Authorization workflow used in the US healthcare system.
- Understood the responsibilities of patients, providers, and payers during authorization requests.
- Explored how medical necessity documentation influences approval decisions.
- Experienced how interactive workflow simulations simplify complex business processes.
