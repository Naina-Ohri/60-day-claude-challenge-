6-month internship preparation roadmap with phases, skills, and weekly tips

6-month internship prep roadmap Click a phase to expand
Month 1–2 · Foundation
DSA + Language Fluency
Arrays & Strings Linked Lists Recursion Python / Java Time Complexity
Weekly goal: Solve 5 LeetCode Easy → Medium problems daily. Focus on arrays, hashmaps, two pointers. Pick one language (Python recommended) and stick to it. Read code from open-source repos to build reading speed.

Month 2–3 · CS Core
Systems + OS + DBMS + Networking
OS Concepts SQL & DBMS Computer Networks OOP + SOLID Git & Linux
Focus: These are asked in every interview — process vs thread, indexing in DB, TCP vs UDP, joins, normalization. Use GeeksForGeeks + InterviewBit for structured revision. Set up a local Postgres DB and write real queries.

Month 3–4 · Build Projects
Hands-on Development + GitHub Portfolio
REST APIs React or Flask GitHub Docker basics Project README
Build 2–3 projects: Aim for one full-stack app (e.g., task manager with login), one data/ML project (e.g., sentiment analyzer), and one automation/API project. Push everything to GitHub with clean READMEs. Recruiters DO look at your profile.

Month 4–5 · Interview Prep
Mock Interviews + System Design Intro
Trees & Graphs DP (Easy-Med) System Design basics Mock interviews Behavioural
Simulate real interviews: Use Pramp, Interviewing.io, or pair with a friend. Practice explaining your thought process out loud — interviewers evaluate communication as much as code. For system design, learn URL shortener, Twitter feed, and rate limiter designs.

Month 5–6 · Apply & Hustle
Applications + Resume + Networking
ATS-friendly resume LinkedIn optimized Cold outreach Internshala / LinkedIn Referrals
Apply aggressively: Send 10–15 applications per week. Customize your cover note per company. Reach out to alumni on LinkedIn — a referral multiplies your callback chances 5×. Track everything in a spreadsheet. Don't wait to feel "ready" — apply now, prep continues.

Non-negotiables: LeetCode streak · Active GitHub · 1 deployed project · LinkedIn with 200+ connections · Resume under 1 page
