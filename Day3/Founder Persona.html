6-month software engineering internship roadmap

Month 1–2
Build the Core
  • Pick 1 language deeply (Python or Java)
  • DSA basics: arrays, strings, linked lists
  • Git + GitHub — commit daily
  • Build 1 small project (CLI / web)
Month 3–4
Get Interview-Ready
  • LeetCode: 2–3 problems/day (Easy → Medium)
  • SQL + basic system design concepts
  • Build a full-stack or API project
  • Polish resume + LinkedIn profile
Month 5–6
Apply & Execute
  • Apply to 10–15 companies per week
  • Mock interviews (peers + AI tools)
  • Prep behavioral stories (STAR method)
  • Contribute to open source (bonus points)
What I actually look for as a hiring manager
Clean, readable code Communication under pressure Problem-solving approach Real projects in portfolio Git hygiene Curiosity + ownership mindset
