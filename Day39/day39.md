# Day 39 – PDF Splitter & Merger

## Project Overview
Built **PDF Splitter & Merger**, a browser-based PDF utility that performs PDF splitting and merging entirely on the client side using HTML, CSS, and JavaScript. The application features a modern UI, drag-and-drop uploads, page previews, responsive layouts, and local PDF processing without uploading files to external servers.

## Key Learnings

- Learned how browser-based PDF processing can be performed securely without uploading files to a server.

- Learned the importance of validation, previews, and progress indicators in productivity applications.
- Discovered how polished UI/UX and smooth animations can make utility software feel like a professional commercial product.
