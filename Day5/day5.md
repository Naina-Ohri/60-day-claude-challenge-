# Day 5 - Context Engineering

## Objective

Learn how providing context improves AI outputs and makes recommendations more personalized.

## Prompt A (Without Context)

Create a 30-day learning roadmap.

Include:

* Weekly milestones
* Daily tasks
* Resources
* Projects
* Final outcome

Make it practical and beginner-friendly.

## Prompt B (With Context)

Context:

* Current Situation: 3rd-year B.Tech IT Student
* Current Skills: C++, Java, Python, SQL, DSA, Canva, Basic AI Tools
* Goal: Land Software Engineering and AI/ML Internships
* Available Time: 2-3 Hours per Day
* Experience Level: Intermediate
* Preferred Learning Style: Projects + Videos

## Comparison

### Which roadmap felt more personalized?

Prompt B (With Context)

### Which roadmap would I actually follow?

Prompt B (With Context)

### What role did context play?

Context helped the AI understand my background, goals, skills, and learning preferences, resulting in a more realistic and actionable roadmap.

## Key Learnings

* Context is often more important than the prompt itself.
* Better context leads to better recommendations.
* AI makes fewer assumptions when provided with relevant information.
* Personalized outputs are more useful and practical.
