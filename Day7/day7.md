# Day 7 - Claude Model Selection & Reasoning Effort

## My Claude Usage Strategy

### Recommended Primary Model

Claude Sonnet 4.6

### Why This Model Fits Me

Claude Sonnet provides the best balance of intelligence, speed, and reasoning quality for coding, learning, AI/ML exploration, resume building, and internship preparation.

### When to Use Haiku

* Quick summaries
* Simple rewrites
* Fast lookups
* Brainstorming ideas

### When to Use Sonnet

* Coding assistance
* Learning technical concepts
* Resume optimization
* Interview preparation
* Daily research

### When to Use Opus

* Deep research
* Project architecture
* Complex problem solving
* Strategic planning

## Recommended Effort Levels

### Low

Quick factual questions and simple tasks.

### Medium

Routine learning and coding support.

### High

Concept learning, debugging, project planning, and career preparation.

### Max

Deep research, difficult DSA problems, and high-stakes decision making.

## Personalized Workflow

| Task                  | Model  | Effort |
| --------------------- | ------ | ------ |
| Coding & DSA          | Sonnet | High   |
| Learning New Concepts | Sonnet | High   |
| Quick Summaries       | Haiku  | Low    |
| Resume & LinkedIn     | Sonnet | High   |
| Deep Research         | Opus   | Max    |
| Project Planning      | Opus   | Max    |

## Key Learnings

* Choosing the correct model is as important as writing a good prompt.
* Sonnet is the best all-round model for most students.
* Higher effort levels improve reasoning quality but should be used selectively.
* Haiku is ideal for speed, while Opus is best for advanced reasoning.
* Using the right model improves productivity and efficiency.
